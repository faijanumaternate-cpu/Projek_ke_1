# Projek_ke_1

## Deskripsi Proyek
Proyek ini dibuat sebagai tugas mata kuliah GAK untuk mempelajari dasar penggunaan Git dan GitHub, mulai dari pembuatan repository, pengelolaan file, hingga proses commit dan push ke GitHub.

## Isi Proyek
- Data_Diri.html : Berisi data diri dalam bentuk halaman HTML.
- README.md : Dokumentasi dan penjelasan singkat mengenai proyek ini.

## Cara Menjalankan Proyek
1. Download atau clone repository ini.
2. Buka file Data_Diri.html.
3. Jalankan file tersebut menggunakan web browser (Chrome, Firefox, atau Edge).
